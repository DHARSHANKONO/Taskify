## Installation Guide

To run Taskify locally, follow these steps:

1. **Clone the Repository:**
   ```bash

   cd Taskify
   ```
2. **Install Dependencies:**
   ```bash
   flutter pub get
   ```
3. **Run the App:**
   ```bash
   flutter run
   ```
