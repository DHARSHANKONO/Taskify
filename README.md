## Installation

To run Taskify on your local machine:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/yourusername/taskify.git
   cd taskify
   ```
2. **Install Dependencies:**
   ```bash
   flutter pub get
   ```
3. **Run the App:**
   ```bash
   flutter run
   ```
